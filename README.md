  Этот плагин позволяет:
  - Выборочно делать открытыми (rel=follow) ссылки на опреледенные домены во всех топиках.
  - Настроить список ИД тех пользователей, в топиках которых ВСЕ ссылки будут открытыми.
  - Заменять ссылки на внешние страницы на внутреннюю "страницу выхода" с возможностью редактирования.
  - Регулирует перманентность формирования ссылок (Если permanent_outredir = false, тогда ссылки формируются динамически и отключение плагина оставит оригинальные ссылки на месте)
  - Для AltoCMS - заменяет ссылки на "страницу выходат" также в доп.полях типа textarea и input
